## Output
一共三种方式，选一个自己觉得喜欢的。设备和设备名称在小喇叭上右键-`声音(s)`-`播放`。`Namew`是黑色字，`Device`是黑色+灰色描述，务必写全！否则就写 N+（N为正整数）慢慢尝试。
